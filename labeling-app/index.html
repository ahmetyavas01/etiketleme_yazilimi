🚀 YOLO Usage Example

1. Google Colab'de YOLOv8 Kurulumu:

!pip install ultralytics
from ultralytics import YOLO
import torch

print(f"PyTorch: {torch.__version__}")
print(f"CUDA Available: {torch.cuda.is_available()}")

2. Dataset ZIP'ini Yükle ve Çıkart:

# Colab'e ZIP dosyasını yükle (Files panelinden sürükle-bırak)
!unzip your_dataset_yolo.zip

# Dataset yapısını kontrol et
!ls -la your_dataset/
!head your_dataset/classes.txt
!head your_dataset/dataset.yaml

3. Model Eğitimi:

# Pre-trained YOLO modeli yükle
model = YOLO('yolov8n.pt')  # nano model (hızlı)
# model = YOLO('yolov8s.pt')  # small model (daha iyi)
# model = YOLO('yolov8m.pt')  # medium model (en iyi)

# Eğitimi başlat
results = model.train(
    data='your_dataset/dataset.yaml',
    epochs=100,                # Epoch sayısı
    imgsz=640,                # Resim boyutu
    batch=16,                 # Batch size
    name='plaka_model',       # Model adı
    save_period=10            # Her 10 epoch'ta kaydet
)

4. Model Testi ve Kullanımı:

# Eğitilen modeli yükle
model = YOLO('runs/detect/plaka_model/weights/best.pt')

# Test resmi üzerinde tespit
results = model('test_image.jpg')
results.show()  # Sonuçları göster

# Birden fazla resim
results = model(['image1.jpg', 'image2.jpg'])
for i, result in enumerate(results):
    result.save(f'result_{i}.jpg')

# Webcam ile real-time tespit
model.predict(source=0, show=True)  # 0 = webcam

5. Model Performansını Değerlendirme:

# Validation seti üzerinde test
metrics = model.val()
print(f"mAP50: {metrics.box.map50}")
print(f"mAP50-95: {metrics.box.map}")

# Confusion matrix ve diğer metrikler
!ls runs/detect/plaka_model/
# confusion_matrix.png, results.png, val_batch0_labels.jpg

6. Model Export (Deployment için):

# ONNX formatına export (hızlı inference)
model.export(format='onnx')

# TensorRT (NVIDIA GPU için)
model.export(format='engine')

# Mobile için
model.export(format='coreml')  # iOS
model.export(format='tflite')  # Android
💡 İpuçları:
• Küçük dataset için epochs=50-100 yeterli
• GPU yoksa yolov8n.pt kullanın (daha hızlı)
• Validation loss azalmayı bırakırsa eğitimi durdurun
• mAP50 > 0.8 ise model başarılı sayılır
